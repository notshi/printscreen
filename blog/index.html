About

Printscreen is a Plated^ component. It started as a simple lua script to allow screenshots to be uploaded to a website.

Back then (2007!) when it was called, 'sBlog', we had to jump through a few hoops to get this to work. Ultimately, we settled on the simplest and most obvious method - sending a message containing the screenshot using a Gmail account. This worked with any blogging service that supported email hosting which included Flickr, Blogger and Livejournal.

Printscreen is on Github
